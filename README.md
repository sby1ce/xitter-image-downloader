# Image Downloader

Download an image from Twitter, Discord and Pixiv in one click with a useful filename. 

## How to build

### For Chrome

```bash
bun install
bun run build
```

### For Firefox

```bash
# you can use bun too but mozilla store uses npm by default
npm install
npm run build:firefox
```
